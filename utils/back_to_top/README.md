**描述**

一个【回到顶部】的按钮，适用于vue的项目中。

借鉴了[vue+element-ui 回到顶部组件](https://blog.csdn.net/caomage/article/details/85006973 "vue+element-ui 回到顶部组件")，谢谢。

---

**功能**

1. 回到顶部。

---

**使用**

```html
<BackToTop></BackToTop>
```

```javascript
<script>
import BackToTop from '@/components/basedata/BackToTop.vue'

    export default {
    	components: {
        	BackToTop
    	}
    }
</script>
```
